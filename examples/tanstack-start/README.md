# netbridge × TanStack Start example

Minimal TanStack Start app whose index route loads data through a **server
function** during SSR — one call via native `fetch` (undici stack) and one via
`axios` (http/https stack). Browser DevTools see neither; netbridge sees both.

## Run

```bash
pnpm install
netbridge -- pnpm dev      # or: pnpm dev:netbridge
```

Open the printed netbridge UI (default http://localhost:4499), then load
http://localhost:3220 — the two upstream `jsonplaceholder.typicode.com`
requests appear with full bodies.

Note: `routeTree.gen.ts` is generated by the TanStack Start Vite plugin on
first `dev` run.
